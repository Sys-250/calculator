# 校园个人简介小主页

一个现代化的校园个人简介网页，展示学生的个人信息、教育背景、专业技能、获奖情况和项目经验。

## 功能特性

- 🎨 **现代UI设计**：渐变紫色主题，卡片式布局
- 📱 **响应式设计**：适配手机和桌面端
- 📊 **数据展示**：GPA、获奖数、项目数统计
- 🔗 **社交链接**：支持主流社交媒体图标
- 📡 **联系方式**：扫码联系功能

## 快速开始

```bash
# 克隆仓库
git clone https://github.com/Sys-250/calculator.git

# 进入目录
cd calculator

# 启动本地服务器
python -m http.server 8000
```

然后访问 `http://localhost:8000` 查看效果。

## 自定义信息

你可以编辑 `index.html` 文件来自定义个人信息：

- 修改头像、姓名、专业
- 更新教育经历
- 添加/修改技能标签
- 更新获奖情况和项目经验
- 更换社交媒体链接
