# 班级静态信息展示系统

一个基于HTML/CSS的班级静态信息展示页面，包含美观的表格展示和交互效果。

## 功能特性

- **表格基础架构**
  - 展示学生基本信息（学号、姓名、性别、年龄、籍贯、专业方向、状态）
  - 支持响应式布局，适配移动端

- **表格美化效果**
  - 隔行变色：偶数行浅灰背景，奇数行白色背景
  - 悬浮效果：鼠标悬停时渐变变色、文字变白、轻微放大、阴影效果
  - 渐变背景和现代化UI设计
  - 性别用彩色圆形标签展示
  - 状态用胶囊式标签区分

## 技术栈

- HTML5
- CSS3

## 使用方法

直接打开 `index.html` 文件即可查看效果。

```bash
# 克隆项目
git clone <repository-url>

# 打开页面
open index.html
```

## 截图预览

页面包含以下模块：
1. 页头：显示班级名称和系统描述
2. 统计卡片：展示总人数、男生、女生、出勤率
3. 学生信息表格：展示详细的学生列表
4. 页脚：版权信息
